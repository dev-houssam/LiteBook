LiteBook est un **lecteur de livres Markdown léger et intelligent** conçu pour lire, organiser et explorer du contenu structuré sous forme de livres.

L'objectif de LiteBook est simple :  
offrir une manière **claire, rapide et programmable** de lire de la documentation, des cours, des livres techniques ou des bases de connaissances.

---

# ✨ Fonctionnalités principales

## 📖 Lecture Markdown

LiteBook permet de lire des fichiers Markdown avec un rendu clair et lisible :

- titres et sections
- listes
- blocs de code
- citations
- images
- diagrammes

Le but est de garder un format **simple, universel et durable**.

---

## 📚 Gestion de livres

LiteBook utilise un format de dossier appelé **`.litebook`**.

Chaque livre est simplement un dossier contenant :

```

mon-livre.litebook
├ book.json
├ chapters/
│   ├ intro.md
│   ├ installation.md
│   └ architecture.md
└ assets/

```

Cela permet :

- une navigation par chapitres
- une organisation claire
- un contenu facile à modifier

---

## 🧭 Navigation rapide

Chaque livre possède une **table des matières**.

Vous pouvez :

- ouvrir un chapitre instantanément
- naviguer entre les sections
- parcourir le livre facilement

LiteBook est pensé pour être **rapide et minimaliste**.

---

## 🤖 Support de fonctionnalités IA

LiteBook peut intégrer des fonctionnalités d'intelligence artificielle grâce à des fichiers de configuration.

Par exemple :

- résumé automatique d'un chapitre
- génération de quiz
- analyse du contenu
- recherche intelligente

Ces fonctionnalités peuvent être définies dans :

```

ai/skills.json

```

---

## 🧩 Extensible

Le moteur de LiteBook est conçu pour supporter des extensions comme :

- rendu mathématique
- diagrammes
- coloration syntaxique
- outils d'analyse
- plugins

LiteBook vise à devenir un **format de livre programmable**.

```
├ chapters/

│   └ chapitre2.md

│   ├ images/
│   └ diagrams/
└ ai/
└ skills.json

```

LiteBook repose sur quelques principes simples :

- **Markdown partout**
- **structure claire**
- **contenu portable**
- **technologie simple**

Un livre LiteBook peut être :

- modifié facilement
- versionné avec Git
- partagé simplement
